# PulseTrack - Health Tracker App ❤️

A Python-based desktop application to track daily health metrics like BMI, steps, sleep, and workouts.

## Features

* User Login & Signup system
* BMI calculation
* Daily health tracking
* Graph visualization (BMI trends)
* Health insights & analysis

## Technologies Used

* Python
* Tkinter (GUI)
* Matplotlib
* JSON (data storage)

## How to Run

1. Install dependencies:
   pip install matplotlib

2. Run the app:
   python HEALTH_TRACKER.py

## Project Idea

This app helps users monitor their daily health habits and visualize progress over time.
